# Volunteer Matcher - פלטפורמה לחיבור בין מתנדבים לקהילה 

פלטפורמה דינמית ורספונסיבית שנועדה לגשר על הפער בין מתנדבים לארגונים חברתיים. הפרויקט פותח במסגרת קורס פיתוח Frontend במרכז האקדמי רופין.

#  [לחצו על הקישור לצפייה באתר החי]
(https://www.google.com/search?q=https://noamaor21.github.io/frontend-capstone-team-4-Volunteer-Matcher/)

# תיאור המערכת

האתר בנוי ממספר דפים המעניקים חווית משתמש מלאה מקצה לקצה:

* **דף הבית:** נקודת הכניסה לאתר המציגה את חזון הפלטפורמה וגישה מהירה לחיפוש עמותות.
* **מאגר עמותות:** ריכוז מידע על עמותות שונות, המאפשר למתנדבים לחקור ולקרוא על פעילויות שונות.
* **דף עמותה מפורט:** דוגמה לדף עומק (עמותת "יד לקשיש") הכולל פעילות ואפשרויות התנדבות.
* **דף הרשמה:** טופס חכם המאפשר למשתמשים חדשים להצטרף למערכת ולהזין את העדפותיהם.
* **אזור אישי:** לוח בקרה למשתמש המציג נתונים אישיים שנשמרו מהטופס.
* **דף אודות:** מידע על המיזם והמטרות החברתיות שלו.

# תכונות מרכזיות (Features)

* **ניהול מצב משתמש (User State):** פיתוח מנגנון השומר את נתוני המשתמש (שם, מיקום, תחומי עניין) באובייקט JSON בתוך ה-LocalStorage.
* **זרימת נתונים:** המערכת קולטת נתונים מטופס ההרשמה ומציגה אותם בצורה מעובדת באזור האישי ללא צורך בבסיס נתונים חיצוני.
* **מצב כהה (Dark Mode):** תמיכה מלאה בכל רכיבי הממשק באמצעות משתני CSS, כולל התאמת תפריטי ניווט ואייקונים.
* **עיצוב רספונסיבי:** התאמה מלאה למסכי מובייל, טאבלט ודסקטופ תוך שימוש ב-Media Queries.
* **נגישות (Accessibility):** עמידה בתקני נגישות הכוללים שימוש בתגיות ARIA והגדרת תוויות (Labels) תקינות.

# טכנולוגיות בשימוש

* **HTML:** שימוש בתגיות סמנטיות למבנה אתר תקני.
* **CSS:** שימוש ב-Variables, Flexbox, Grid ו-Bootstrap .
* **JavaScript:** ניהול אירועים, מניפולציה של ה-DOM וניהול LocalStorage.
* **Git & GitHub:** ניהול גרסאות ועבודה צוותית מבוססת ענפים .

# הוראות הפעלה

1. בצעו Clone לפרויקט למחשב המקומי.
2. פתחו את הקובץ `index.html` באמצעות הדפדפן (או באמצעות Live Server ב-VS Code).
3. ניתן לעבור בין הדפים דרך סרגל הניווט העליון.



# צוות הפרויקט

* **דניאל פרזן:** אחראי על האזור האישי, טופס ההרשמה ולוגיקת הנתונים (JSON).
* **נועה מאור:** אחראית על דף הבית, סרגל הניווט (Navbar).
* **גילי אמסלם:** אחראית על דף המאגר, דף עמותת "יד לקשיש" ועיצוב ה-UI.
* **שחף גלאם:** אחראית על דף אודות (About) ועיצוב ה-Footer.
