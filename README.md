Volunteer Matcher - פלטפורמה לחיבור בין מתנדבים לקהילה
פלטפורמה דינמית ורספונסיבית שנועדה לגשר על הפער בין מתנדבים לארגונים חברתיים. הפרויקט פותח במסגרת קורס פיתוח Frontend במרכז האקדמי רופין.

תכונות מרכזיות בפרויקט
במהלך הפיתוח הושם דגש על שלושה עקרונות מרכזיים: חווית משתמש, לוגיקה הנדסית ונגישות.
ניהול מצב משתמש: פיתוח מנגנון השומר את נתוני המשתמש (שם, מיקום, תחומי עניין) באובייקט בתוך ה-LocalStorage.
זרימת נתונים: המערכת קולטת נתונים מטופס ההרשמה ומציגה אותם בצורה מעובדת באזור האישי ללא צורך בבסיס נתונים חיצוני.

עיצוב ומצב כהה (Dark Mode):
תמיכה במצב כהה: התאמה מלאה של כל רכיבי ממשק המשתמש למצב כהה באמצעות משתני CSS, כולל התאמת תפריטי ניווט ואייקוני SVG.

עיצוב רספונסיבי: התאמה מלאה למסכי מובייל, טאבלט ודסקטופ תוך שימוש ב-Media Queries.

נגישות (Accessibility)

עמידה בתקני נגישות הכוללים שימוש בתגיות ARIA, הגדרת תוויות (Labels) תקינות.

טכנולוגיות בשימוש:
HTML5: שימוש בתגיות סמנטיות למבנה אתר תקני.

CSS: שימוש ב-Variables, Flexbox ו-Grid.

JavaScript : ניהול אירועים, פונקציות חישוב ומניפולציה של ה-DOM.

Git & GitHub: ניהול גרסאות ועבודה צוותית מבוססת ענפים (Branches).
